# Live Object Model Reference

Comprehensive reference for the Ableton Live Object Model (LOM) — the object hierarchy exposed by Live's
Python runtime to Control Surface scripts, Max for Live devices, and external clients.

## About

Ableton does not publicly document the Live Python API. This reference synthesizes three sources into a
single accurate resource:

- **API stubs** — auto-generated by running introspection inside Live. Complete class/member inventory.
- **Max for Live docs** — richer descriptions and gotcha coverage from the Max documentation.
- **Direct probing** — runtime experiments to confirm behavior that neither source clarifies.

## How to Read

Each page documents one LOM class with:

- **Summary tables** — quick overview of all children, properties, and methods
- **Detail sections** — per-member descriptions with type info, quirks, and limitations
- **Open Questions** — unresolved behavior flagged for future investigation

Use the sidebar navigation to browse by LOM hierarchy, or search for a specific class or member.
